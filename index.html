Component Property Description Unit Types Example
Content
  • Width
  • Height
 Defines the size of the content area. Does not include padding, border, or margin.
  • px (pixels)
  • % (percent relative to the parent element)
  • em / rem (relative to parent element font size / root element font size)
  • vw / vh (viewport width / viewport height)
 width: 200px;
height: 300px;
Padding
  • padding
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
 Creates space between the content and the border.
  • px
  • %
  • em / rem
 padding: 15px;
padding-left: 20px;
Border
  • border
  • border-width
  • border-color
  • border-style
  • border-radius
 Defines the edge around the padding. Can be styled with different widths, colors, and styles.
  • px
  • % (for border radius)
  • em / rem
 border: 2px solid black; border-radius: 10px;
Margin
  • margin
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
 Creates space between the border and the adjacent elements.
  • px
  • %
  • em / rem
  • auto (for centering)
 margin: 20px; margin-top: 10px;
